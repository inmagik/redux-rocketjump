Promise Plugin

This plugin adds Promises to action dispatchers using the awesome library redux-saga-thunk.

import rjWithPromise from 'redux-rocketjump/plugins/promise'

const {
    actions: {
        load,
        unload,
    },
    saga,
} = rj(
      rjWithPromise,
      {
          type: 'GET_TODOS',
          state: 'todos',
          api: loadTodosFromApi,
      }
    )()

// ... next ...
store.dispatch(load())
  .then(() => /* ... */)
  .catch(() => /* ... */)

In order to work, this plugin requires you to install redux-saga-thunk
